# BECCA Learning Hub

This repository contains a Quarto website for the Best Catch Assessment (BECCA) Learning Hub.

## Local preview

Install Quarto from https://quarto.org, then run:

```bash
quarto preview
```

## Render the site

```bash
quarto render
```

The rendered website will be written to the `docs/` folder.

## GitHub Pages setup

In GitHub, go to:

`Settings > Pages > Build and deployment`

Use either:

1. **GitHub Actions**, using the included workflow; or
2. **Deploy from branch**, then select `main` and `/docs` after rendering locally.

## Suggested repository name

`becca-learning-hub`

Default GitHub Pages URL:

`https://project-seagrass.github.io/becca-learning-hub/`

Optional custom domains:

- `becca.projectseagrass.org`
- `learn.projectseagrass.org/becca`
